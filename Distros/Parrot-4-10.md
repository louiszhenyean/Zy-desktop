# Problem Fixed 

### Fix Square Fonts
- `System Settings > Application Style > GNOME/GTK Application Style > Sans Serif 10`
- Reference: https://community.parrotsec.org/t/parrot-4-8-bug-report/12224/15  

### Emoji Fonts
- `sudo apt reinstall fonts-noto-color-emoji`
