### Install Devilspie
- [How to be cool and make Visual Studio Code (VSCode) transparent][https://dev.to/emmanuelnk/how-to-be-cool-and-make-vscode-transparent-56ib]
- `sudo apt-get install devilspie`
- `mkdir -p ~/.devilspie`
- `cp vscode.ds ~/.devilspie/vscode.ds`
